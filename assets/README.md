# Assets

`assets/design-pdf-extract/` is the canonical source set. Its filenames are the naming authority for the rest of the asset tree.

`assets/images/web/` contains production WebP files used by the site. Those files now reuse the canonical `design-pdf-extract` basenames.

`assets/images/` contains a small set of older raster exports that could be matched confidently to the canonical source names.

`assets/images/legacy/` contains older exports that are currently unused by the site and were not renamed because the match to a canonical source file was not strong enough to do safely.
